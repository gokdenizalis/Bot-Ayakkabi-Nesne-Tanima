# Bot ve Ayakkabı Nesne Tanıma Projesi

Bu proje, YOLOv8 algoritması kullanılarak derin öğrenme tabanlı nesne tespiti yapılması amacıyla geliştirilmiştir.

## Proje Detayları
- **Veri Seti:** 1000 adet Bot ve 1000 adet Ayakkabı görseli.
- **Model:** YOLOv8 Nano
- **Amaç:** Ayakkabı ve bot türlerini yüksek doğrulukla birbirinden ayırt etmek.

## Dosya İçerikleri
- `test_yap.py`: Modelin eğitimini başlatan ana Python scripti.
- `argümanlar.yaml`: Eğitimin teknik parametrelerini (epoch, batch size, learning rate vb.) içeren dosya.
